# 100th-Kindergarten-of-Athens
ΑΠΟΘΕΤΗΡΙΟ 100ου ΝΗΠΙΑΓΩΓΕΙΟΥ ΑΘΗΝΩΝ ΓΙΑ ΤΟΝ ΔΙΑΓΩΝΙΣΜΟ ΑΝΟΙΧΤΩΝ ΤΕΧΝΟΛΟΓΙΩΝ

Τίτλος:«Μικρές Πράξεις, Μεγάλες Αλλαγές στη Γειτονιά μας»

Το 100ο Νηπιαγωγείο Αθηνών είναι ένα μικρό σχολείο στο κέντρο της Αθήνας, που επιδιώκει να αποτελεί ενεργό μέλος εκπαιδευτικών κοινοτήτων που ενισχύουν την επικοινωνία, τη συνεργατικότητα, την ανταλλαγή καλών πρακτικών και την καλλιέργεια δεξιοτήτων. Τα τελευταία χρόνια συμμετέχει ανελλειπώς δυναμικά σε προγράμματα eTwinning, τα οποία έχουν μάλιστα αποσπάσει σημαντικές διακρίσεις. Παράλληλα, αναπτύσσει ουσιαστική δράση με βασικό άξονα τη φιλοσοφία της αειφορίας, λαμβάνοντας μέρος και στο Δίκτυο Οικολογικών Σχολείων.
Πέρα από τη σύνδεση με το φυσικό περιβάλλον, το σχολείο μας δίνει ιδιαίτερη έμφαση στο κοινωνικοσυναισθηματικό κλίμα που καλλιεργείται στην τάξη, καθώς και στη δημιουργία ενός ασφαλούς και υποστηρικτικού περιβάλλοντος, όπου όλα τα παιδιά έχουν τη δυνατότητα να εκφραστούν ισότιμα. Κεντρικός στόχος είναι η διαμόρφωση χαρούμενων, ενεργών και υπεύθυνων πολιτών, με την ενεργό  δράση να αποτελεί αναπόσπαστο στοιχείο της καθημερινότητάς τους. Στο πλαίσιο αυτής της παιδαγωγικής προσέγγισης, υλοποιούνται δράσεις στις οποίες συμμετέχουν με ιδιαίτερο ενδιαφέρον και οι γονείς.
Μικροί και μεγάλοι βιώνουμε καθημερινά στη γειτονιά μας, στα Πατήσια, διάφορες καταστάσεις, που αφενός υποβαθμίζουν την ποιότητα της ζωής μας αφετέρου μας προβληματίζουν ιδιαίτερα. Έτσι λοιπόν αποφασίσαμε να δράσουμε ξανά, στέλνοντας μήνυμα ηχηρό στη γειτονιά μας, με την ελπίδα πως σε αυτή τη προσπάθεια θα έχουμε αρκετούς συνοδοιπόρους. 

Σε πρώτη φάση, θα βγούμε με τα παιδιά από το σχολείο, ώστε να κάνουμε έρευνα πεδίου. Πιο συγκεκριμένα, θα περπατήσουμε στη γειτονιά μας, με τελικό προορισμό το πάρκο ΦΙΞ  που βρίσκεται πολύ κοντά στο σχολείο μας. Κατά τη διάρκεια του περιπάτου, οι μαθητές θα καταγράψουν και θα φωτογραφίσουν με tablet  πιθανά προβλήματα που θα συναντήσουν. Μερικά πιθανά προβλήματα στα οποία  θα εστιάσουμε, είναι η σπατάλη νερού σε διάφορα σημεία, η διαχείριση απορριμμάτων και ο έλεγχος για τους σωστούς κάδους. Επίσης, θα ελέγξουμε την προσβασιμότητα που διατίθεται στη γειτονιά για άτομα με διάφορες αναπηρίες καθώς και για άλλες κοινωνικές ομάδες ( πχ. γονείς με καρότσι μωρού, ηλικιωμένοι με καροτσάκια για τα ψώνια τους κτλ). Τέλος, θα διερευνήσουμε τη μέριμνα για τα αδέσποτα ζώα της γειτονιάς μας. 
Στη συνέχεια, με όλες τις πληροφορίες που θα έχουμε συλλέξει, θα φτιάξουμε έναν εννοιολογικό χάρτη ώστε να αποτυπωθούν. Επιπλέον θα χαρτογραφήσουμε τη γειτονιά μας, ΄΄σημαδεύοντας΄΄ τα σημεία που παρουσιάζονται προβλήματα, ώστε να επανέλθουμε στο τέλος της δράσης κάνοντας την αποτίμηση. 
Η επόμενη φάση θα περιλαμβάνει διεξοδική συζήτηση που θα περιλαμβάνει αναζήτηση προτάσεων και λύσεων. Οι μαθητές, θα έρθουν σε επαφή με την τεχνητή νοημοσύνη και θα ζητήσουν λύσεις για τα προβλήματα που ανακάλυψαν. Στη συνέχεια, οι λύσεις που προτάθηκαν απο τους μαθητές αλλά και από το εργαλείο ΤΝ, θα μεταφερθούν σε ψηφιακή φόρμα ψηφοφορίας, η οποία θα συμπληρωθεί από μαθητές, εκπαιδευτικούς και γονείς ώστε να καταλήξουμε σε αποφάσεις. 
Οι αποφάσεις μας, θα γίνουν μηνύματα προς όλους. Στους γονείς των μαθητών, στους κατοίκους της περιοχής, στον Δήμαρχο Αθηναίων, σε γειτονικά σχολεία, ωστε η φωνή μας να ακουστεί και όλοι μαζί να προσπαθήσουμε να αλλάξουμε τη γειτονιά μας. Τα μηνύματα, θα γραφτούν και θα εικονογραφηθούν από τα παιδιά και θα εκτυπωθούν σε ειδικά βιοδιασπώμενα χαρτιά με σπόρους, ώστε κάθε ένας που θα το πάρει στα χέρια του, να το φυτέψει κάπου στη γειτονιά μας. Οι ίδιοι οι μαθητές, ως μικροί αγγελιοφόροι, θα βγουν από το σχολείο ξανά και θα μοιράσουν τα φυλλάδιά τους. 
Τέλος, θα γίνει ανοιχτό κάλεσμα στο σχολείο μας για τους γονείς των μαθητών μας, ώστε να έχουν και εκείνοι την ευκαιρία με τη σειρά τους να αποτυπώσουν τα δικά τους μηνύματα στα χαρτιά. 
Η σκέψη μας είναι, πως μια φιλοσοφία, δεν πρέπει να μένει κλειστή μέσα σε ένα μυαλό ή σε μία τάξη. Τα παιδιά, ως μικροί πολίτες του μέλλοντος, πιστεύουμε πως θα πρέπει να αντιληφθούν ότι είναι αναγκαίο να είμαστε ενεργοί και να συμβάλλουμε για τις αλλαγές που θέλουμε να φέρουμε στη ζωή τη δική μας και των γύρω μας. 
Από τις δράσεις που θα υλοποιηθούν και τα μηνύματα των παιδιών, θα δημιουργηθεί ένα ηλεκτρονικό αποθετήριο με audiobooks, φτιαγμένα από τα παιδιά, με τα οποία θα επιτευχθεί επιπλέον διάχυση των στόχων μας.
